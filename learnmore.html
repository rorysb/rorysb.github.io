This page hopes to give further information on the project performed over the course of my work experience.

Overview

The project underwent concerned the use of TensorFlow, Google's own technology, for the creation of results similar to those gained by Google in their deepdream project as well as working to train a network using my own set of training data.

Setting up the server for work

A very important part of the project was the creation and set up of the server. Amazon's server service was used and using scripts from my mentor Gavin Gray I setup the server with Docker, a tool I found very useful, allowing for easy reattempts at various tasks throughout my period of work experience.

Creating images similar to those of deepdream

Using notebooks provided by Google for use of their TensorFlow software I experimented creating images similar to those created by Google. I also worked to condense the code within the supplied notebook into my own, this as an exercise to better understand the code that was provided. Doing this provided me with information allowing me to further experiment with the alteration of imagery by changing the number of 'octaves' (a label given by Google) as well as the number iteration run at each 'octave'. As I believe the octaves refer to the number of times the image is edited, at each stage creating more seemless imagery by stitching together the cut up image.
